📈 Stock Market Prediction Using Social Media Sentiment Analysis and Natural Language Processing

📖 Overview

This project presents a machine learning approach for predicting stock market trends by combining historical stock price data with sentiment extracted from social media posts. The objective is to analyze how public sentiment influences stock price movements and leverage both textual and financial data to build predictive models.

The project covers the complete machine learning workflow, including data preprocessing, exploratory data analysis, sentiment analysis, feature engineering, model development, visualization, and evaluation.

---

 ✨ Features

- 📊 Historical stock market analysis
- 💬 Social media sentiment analysis using NLP
- 🧹 Data preprocessing and feature engineering
- 📈 Exploratory Data Analysis (EDA)
- 🤖 Machine Learning-based stock prediction
- 📉 Performance evaluation and visualization

📁 Dataset

The project utilizes two datasets:

- **stock_tweets.csv** – Stock-related tweets used for sentiment analysis.
- **stock_yfinance_data.csv** – Historical stock market data collected using Yahoo Finance.

🚀 Installation

Clone the repository

```bash
git clone https://github.com/Shewe666/Predicting-Stock-Market-Using-Social-Media-Sentiment-Analysis-and-Natural-Language-Processing.git
```

Move to the project directory

```bash
cd Predicting-Stock-Market-Using-Social-Media-Sentiment-Analysis-and-Natural-Language-Processing
```

Install the required dependencies

```bash
pip install -r requirements.txt
```

---

▶️ Running the Project

1. Open the notebook inside the `notebooks` folder.
2. Ensure both datasets are present inside the `data` folder.
3. Run all notebook cells sequentially.
